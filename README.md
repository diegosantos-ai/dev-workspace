# DevOps Workspace Central

Este repositório serve como hub central para automação, infraestrutura como código (IaC), configurações locais (dotfiles) e documentação operacional de práticas DevOps.

## 📂 Estrutura do Repositório

- **`rotina-devops.md`**: Guia principal das rotinas diárias e processos padrão.
- **`docs-referencia/`**: Documentações, guias e links úteis.
- **`dotfiles/`**: Configurações de ambiente pessoal (ex: configuração de VS Code e MCP).
- **`ovh-terraform/`**: Configurações de provisionamento via Terraform focadas na OVH Cloud.
- **`playbooks/`**: Checklists processuais (rotinas da manhã, setup de novas máquinas, PPOs).
- **`scripts/`**: Scripts de shell para automação e bootstrap (ex: `setup-machine.sh`).
- **`templates/`**: Bases arquiteturais reutilizáveis, incluindo implementações completas como o `terraform-aws-base`.
- **`terraform-test/`**: Ambiente de sandbox para testes e validações rápidas de módulos Terraform.
- **`teste-python/`**: Ambiente para pequenos scripts de utilidade e testes rápidos de configuração.

## 🚀 Propósito

Centralizar o conhecimento, automatizar tarefas repetitivas e fornecer um estado base rapidamente reprodutível para as operações diárias de engenharia, garantindo alinhamento e eficiência no fluxo de trabalho.
