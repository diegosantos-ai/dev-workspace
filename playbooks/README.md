# Módulo: Playbooks (Roteiros Operacionais)

Este diretório centraliza documentos de **execução passo-a-passo** (runbooks, checklists e procedimentos). Diferente do `docs-referencia/` (que é normativo), este módulo é focado na ação imediata.

## 📂 Conteúdo Principal

- `checklist-onboarding.md`: Fluxo de 10 passos para setup de máquina nova.
- `manutencao.md`: Procedimentos de limpeza e atualização de ambiente.
- `checklist-manha.md`: Rito de início de expediente.
- `governanca-ia.md`: Regras de conduta para interação com agentes.

## 🎯 Regra de Uso
Se o documento contém comandos para copiar e colar ou uma sequência de ações `[ ]`, ele deve residir aqui.
